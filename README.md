BoxMonitor
==========

A realtime Linux system monitoring, using node.js with the Sails framework and d3.js for charting results.

Features
--------

- System informations
- CPU monitoring
- RAM overview
- Disks usage
- And more to come...

Requirement
-----------

- Node.js

Installation
------------

> $ git clone https://github.com/emericg/BoxMonitor.git  
> $ npm install  
> $ node apps  

Then browse http://localhost:1337/
